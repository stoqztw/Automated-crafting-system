# AutoCraft ⚙️

Система автоматизированного крафта для Minecraft с использованием мода ComputerCraft.

## Описание 📝

AutoCraft — это Lua-скрипт, предназначенный для запуска на компьютере (Computer) в Minecraft с установленным модом ComputerCraft. Скрипт обеспечивает автоматизацию крафтинга через интерфейс монитора и взаимодействие с ящиками (Extended Drawers).
